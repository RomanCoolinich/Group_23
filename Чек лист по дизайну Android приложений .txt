1. Приложение не имеет  кнопку “назад” 
2. Минимальная размер элементов 48х48 dp
3. Приложение не использует нижнюю панель с менее чем двумя функциональными кнопками
4.  При использовании клавиатуры нижняя панель скрывается
5. Приложение не использует жесты на переднем слое для доступа к заднему слою
6. Контент переднего слоя при прокрутке не перекрывает задний слой
7. В приложение может отображаться только один баннер
8. Текст отклоняющего действия в баннерных сообщениях расположен слева, а подтверждающее справа
9. Баннерное сообщение не содержит ссылок
10.  Ширина кнопок больше длины текста кнопок
11.  Назначение кнопок нижней навигации содержит соответствующую иконку
12.  Приложение использует чекбоксы для выбора элементов списка
13.  В приложении используются индикаторы процесса для отображения состояния приложения (загрузка, открытие...)
14. Все текстовые поля приложения содержат плейсхолдеры, которые занимают не более одной строки
15.  Всплывающие подсказки не отображаются более 1,5 секунд
16.  Панель навигации находиться с левой стороны если в приложении не установлен язык с письмом справа налево.
