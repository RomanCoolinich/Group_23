 1. Создать файл “new.json”                                                       >new.json
 2. Добавить файл под гит.                                                        git add new.json                 
 3. Закоммитить файл.                                                             git commit new.json -m 'new.json was added'
 4. Отправить файл на внешний GitHub репозиторий.                                 git push
